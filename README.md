# API de Cadastro e Segurança de Usuários 🛡️

Este é um projeto Backend em desenvolvimento focado na criação de uma API para gerenciamento e autenticação de usuários. A aplicação está sendo construída com **Java** e **Spring Boot**, visando aplicar boas práticas de engenharia de software e segurança.

## 🎯 Objetivo do Projeto
Estruturar um sistema de cadastro seguro e eficiente, isolando o modelo de domínio do tráfego de dados externo e implementando filtros de segurança para controle de acesso.

## 🛠️ Tecnologias Utilizadas
* **Java** * **Spring Boot** (Web, Security)
* **Spring Security** (Configuração de rotas e filtros de acesso)
* **H2 Database** (Banco de dados em memória para testes e prototipagem ágil)
* **Padrão DTO (Data Transfer Object)** (Para segurança e organização na transferência de dados)

## 🚧 Status
Em desenvolvimento. Próximos passos incluem a implementação completa de autenticação baseada em tokens (JWT) e validação de dados de entrada.
